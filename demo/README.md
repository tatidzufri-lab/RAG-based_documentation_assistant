# Демонстрация работы ассистента

Эта папка содержит скриншоты, демонстрирующие работу RAG-ассистента по документации и результаты его оценки.

## Описание скриншотов

### 1. `ragas_metrics_explanation.png`
**Метрики оценки и пояснения**

Этот скриншот показывает:
- Результаты оценки системы по метрикам RAGAS
- **Faithfulness (Точность ответа)** — насколько ответ соответствует предоставленному контексту (0.0–1.0)
- **Context Precision (Точность контекста)** — качество извлечённого контекста для ответа на вопрос (0.0–1.0)
- Примеры оценок для конкретных вопросов
- Пояснения к каждой метрике

### 2. `ragas_evaluation_results.png`
**Детальные результаты оценки**

Этот скриншот демонстрирует:
- Полный процесс работы ассистента: поиск документов, формирование промпта, генерация ответа
- Детальные результаты оценки по каждому вопросу из тестового набора
- Общий средний балл системы: **0.9000** (отличное качество)
- Примеры вопросов, на которые отвечает ассистент:
  - Вопросы о трансфере и логистике
  - Вопросы о составе группы
  - Вопросы о программе и методологии
  - Вопросы о рекомендациях по экипировке

## Что демонстрируют скриншоты

Скриншоты показывают, что система:
- ✅ Успешно обрабатывает вопросы пользователей
- ✅ Находит релевантные документы в векторном хранилище
- ✅ Генерирует точные ответы на основе найденного контекста
- ✅ Показывает высокое качество работы (Faithfulness: 1.0000, Context Precision: 0.8000)
- ✅ Получает оценку "Отличное качество!" по метрикам RAGAS

## Как получить такие результаты

Для запуска оценки системы используйте скрипт `evaluate_ragas.py` в папке `assistant_api`:

```bash
cd assistant_api
python evaluate_ragas.py
```

Система автоматически:
1. Загрузит тестовые вопросы
2. Обработает каждый вопрос через RAG pipeline
3. Оценит качество ответов с помощью метрик RAGAS
4. Выведет детальные результаты и общую статистику
